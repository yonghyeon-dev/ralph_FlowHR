# L1: Platform Console (플랫폼 운영)

## 범위
플랫폼 대시보드, 테넌트 관리, 빌링, 서포트, 모니터링, 감사, 설정

## 스크린
- PC-001: 운영 대시보드 (KPI + 큐 + 헬스 + 시그널)
- PC-101~104: 테넌트 관리 (목록 + 상세)
- PC-201~205: 플랜 & 빌링 (MRR + 플랜 카탈로그 + 인보이스)
- PC-401~403: 서포트 (티켓 큐 + SLA)
- PC-501~502: 모니터링 (API + 업타임)
- PC-601~604: 감사 & 보안
- PC-701~702: 플랫폼 설정
