# L1: Employee Portal (직원 포탈)

## 범위
직원 홈, 스케줄, 신청, 수신함, 문서, 프로필

## 스크린
- TE-001~002: 홈 (데스크톱 + 모바일)
- TE-101~103: 출퇴근 + 주간 스케줄 + 이력
- TE-201~205: 신청 유형 + 휴가 폼 + 정정 폼 + 이력
- TE-301~302: 알림 수신함 + 상세
- TE-401~403: 문서 서명 + 뷰어 + 보관함
- TE-501~503: 프로필 (기본정보 + 휴가잔여 + 성과)

## WI
- WI-060: 직원 홈
- WI-061: 출퇴근 체크 + 주간 스케줄
- WI-062: 출결 이력
- WI-063: 신청 유형 + 휴가 신청 폼
- WI-064: 근태 정정 폼 + 신청 이력
- WI-065: 알림 수신함
- WI-066: 직원 문서 (서명 + 뷰어 + 보관함)
- WI-067: 직원 프로필
