## 범위

## L2 모듈

- **디자인 시스템**: 토큰(컬러, 타이포, 스페이싱) + 기본 컴포넌트(Button, Card, Badge, Input, Table, KPICard, Chart)
- **레이아웃**: AppShell (Header, Sidebar, Main), AdminSidebar, EmployeeSidebar, PlatformSidebar, Drawer, Modal, Toast
